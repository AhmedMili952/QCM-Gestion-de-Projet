QCM XTI103 — Git & GitHub

1. Quelle commande permet de créer une branche dev ?

La commande git branch dev crée une nouvelle branche sans s’y déplacer.

2. Qu’est-ce qu’une branche Git ?

Une branche est un pointeur vers une suite de commits représentant une ligne de développement.

3. Qu’est-ce qu’un commit Git ?

Un commit est un snapshot du projet à un instant donné avec un message et un hash.

4. Que fait la commande git merge ?

git merge combine l’historique d’une branche source dans la branche courante.

5. Que représente HEAD dans Git ?

HEAD pointe vers le commit courant, qui sera le parent du prochain commit.
